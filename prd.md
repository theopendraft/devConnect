Product Requirements Document (PRD)
Project Name: DevConnect (MVP)
Objective: To build a simple, functional social media web application that demonstrates AI-assisted development, modern tech stack implementation, and a complete CI/CD pipeline within a 3-day deadline.

1. Target Audience

Technical Assessors evaluating code structure, deployment pipelines, and AI tool utilization.

Mock Users (Developers) looking to post short text updates.

2. Scope of Work (MVP)
To ensure the project is completed and deployed flawlessly within the deadline, we will strictly focus on core functionalities.

In-Scope: User Authentication (Login/Register), Create a Text Post, View a Global Feed of Posts.

Out-of-Scope (Future Iterations): Image/Video uploads, Comments, Likes, User Profiles, Follower systems, Direct Messaging.

3. User Stories

As a new user, I want to sign up and log in securely so that my identity is verified.

As an authenticated user, I want to write and publish a text post so that others can read my thoughts.

As a visitor, I want to see a global feed of the latest posts chronologically (newest first).

4. UI/UX Requirements

Design System: Clean, minimalist, and dark-mode preferred (developer-centric).

Responsiveness: Must work seamlessly on both mobile and desktop screens.

Layout:

Navbar: Logo on the left, Login/Logout button on the right.

Main Content Area: A central column containing a "Create Post" input box at the top, followed by a vertically scrolling list of post cards.
